# activeTAPI-DyNav v2013

Here you will find the latest setup and tools for activeTAPI-DyNav v2013.

## Introduction

### Installation
The first step is to download and install [activeTAPI.COM-DyNav_v2013_Setup.msi](). 

![C:\Users\Public\Documents\Sync\Daten\_bitbucket\activeTAPI.v2013\Navision\src\SetupTools\doc\img\activeTAPI-DyNav.v2013_installed.png](img/activeTAPI-DyNav.v2013_installed)

Once this is done, you can test the proper setup and try to connect with your telephony.

* Connect with Windows-TAPI (Telephony Hardware Connection Test)

If all this works fine you can continue and install the [NAV-Objects](<https://github.com/SchmidteServices/activeTAPI-DyNav/tree/master/NavObjects>) in you NAV instance.
